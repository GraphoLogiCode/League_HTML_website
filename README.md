# Dookbeggy-2-

A simple educational HTML website about League of Legends (LoL) and gaming terminology.

## About This Project

This is a personal project that provides information about:
- **LOL**: The internet slang acronym and its meanings
- **League of Legends**: The popular MOBA game by Riot Games
- **CS (Creep Score) Farming**: Essential gameplay mechanics
- **Champion Skills**: How champion abilities work in League of Legends

## Project Structure

```
.
├── index.html          # Main page introducing LOL acronym
├── 1.html             # League of Legends game information
├── 2.html             # CS (Creep Score) Farming guide
├── 3.html             # Champion abilities and skills
├── LoL1.gif           # League of Legends animated image
├── Web/               # Assets directory
│   ├── style.css      # Stylesheet for the website
│   └── ...            # Additional HTML files
└── README.md          # This file
```

## How to View

1. Open `index.html` in your web browser to start exploring
2. Navigate through the site using the menu links on each page
3. No server setup required - all files are static HTML

## Features

- Simple, clean navigation between pages
- Responsive design with CSS Grid layout
- External links to relevant resources (e.g., op.gg for League of Legends stats)
- Educational content about gaming terminology and mechanics

## Technologies Used

- HTML5
- CSS3 (with Grid layout and media queries)
- No JavaScript - pure static HTML site
